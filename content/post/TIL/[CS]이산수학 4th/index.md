# 언어

> 기본요소들의 집합으로 만들어지는 문자열

* 알파벳 : 기호들의 집합
* 문법 : 알파벳으로부터 문장들의 집합을 형성하는 규칙이 존재
* 의미론 : 규칙에 합당하게 만들어진 문장들이 어떤 의미를 갖는지 결정

## 1. 형식언어

구성요소 : 심벌, 알파벳, 문자열, 공문자열

## 2. 구-구문 문법

`G = (V, T, S, P)` 👉 `V : 기호의 집합` `T : 단말기호` `S : 시작기호` `P : 생성규칙`
